## 二、M-01：数据加载与解析

### TC-M01-001: 标准二进制文件加载

**对应需求**：F-001

**前置条件**：
- 存在符合协议规范的 .raw/.bin 二进制文件
- 文件包头为 A5A5 A5，数据头为 AABB

**测试步骤**：
1. 点击"打开"按钮或使用菜单「文件→打开」
2. 选择标准格式的二进制文件
3. 观察进度条和状态提示
4. 加载完成后检查表格数据

**预期结果**：
- 进度条正常显示 0%→100%
- 状态文本显示"正在解析数据..."
- 耗时计时器实时更新（HH:MM:SS 格式）
- 解析完成后弹框提示成功，显示解析数量和耗时
- 表格正确显示解析后的数据记录

**优先级**：P0

### TC-M01-009: 消息类型自动识别

**对应需求**：F-015

**前置条件**：
- 包含 BC→RT、RT→BC、RT→RT、Broadcast 四种类型的测试数据

**测试步骤**：
1. 加载混合消息类型的测试数据

- T_R=1 → RT→BC
