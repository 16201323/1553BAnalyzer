# 1553B 数据智能分析工具

基于 Qt 开发的 1553B 总线数据智能分析工具，支持数据解析、筛选、可视化、AI 查询等功能。

## 功能特性

### 数据加载与解析
- 支持 1553B 二进制数据文件（.raw/.bin/.data）
- 并行解析，支持大文件快速加载
- 包头、数据头、命令字等完整校验
- 支持大端/小端字节序

### 数据展示
- 表格展示，支持分页（50/100/200/500/1000 条/页）
- 表头排序（升序/降序）
- 错误行高亮显示
- 16 进制源码查看

### 数据筛选
- AI 自然语言筛选（支持大模型）
- 算式筛选（支持 AND/OR 逻辑组合）
- 筛选列名特殊显示（加粗蓝色 + 筛选图标，悬停显示筛选算式）
- 筛选弹框记忆上次输入的算式
- 筛选过程显示进度条（异步筛选，不阻塞 UI）
- 数据范围选择（所有数据/当前页/当前筛选）

### 数据可视化
- **甘特图**：时:分:秒格式时间轴，支持缩放
- **饼图**：终端地址分布、消息类型分布、通道状态分布
- **柱状图**：数据量统计
- **折线图**：时间趋势分析
- **周期间隔折线图**：指定 RT 和子地址的时间间隔变化趋势，支持数据点数值标注和 CSV 导出

### 数据导出
- CSV 格式导出
- PDF 格式导出（含分析报告）
- 周期间隔分析 CSV 导出（含序号、RT、子地址、数据包时间、间隔时间）
- 导出进度条显示

### AI 智能分析
- 支持 Kimi、Ollama 等大模型
- 自然语言查询
- 智能分析报告生成
- 周期间隔分析（输入"显示终端地址X且子地址Y的周期间隔"自动生成折线图）
- 分析模式响应无多余空行

### 数据库存储
- SQLite 数据库持久化
- 列式存储优化查询性能
- 支持大数据量（>10 万条）自动切换数据库模式

## 技术栈

- **框架**: Qt 5.9.9
- **语言**: C++14
- **编译器**: MSVC 2015 (Visual Studio 2015)
- **数据库**: SQLite 3
- **AI 集成**: RESTful API
- **语音识别**: Vosk（离线语音识别引擎）
- **构建工具**: qmake / CMake

## 编译步骤

### Windows (MSVC 2015)

```bash
# 使用 qmake（推荐）
qmake 1553BAnalyzer.pro -spec win32-msvc2015
nmake

# 或使用 CMake
mkdir build && cd build
cmake .. -G "Visual Studio 14 2015 Win64"
cmake --build . --config Release
```

### Linux

```bash
mkdir build && cd build
qmake ..
make -j$(nproc)
```

## 使用说明

### 数据加载
1. 点击"打开"或拖拽文件到窗口
2. 选择 1553B 二进制文件
3. 等待解析完成（大文件会自动使用数据库模式）

### 数据筛选
1. 右键点击列头选择"按此值筛选"
2. 在弹框中输入筛选算式（如 `>5&&<20`），支持 AND(&&)/OR(||) 逻辑组合
3. 已筛选的列名会加粗蓝色显示，悬停可查看当前筛选算式
4. 再次打开筛选弹框会自动显示上次输入的算式
5. 筛选过程显示进度条，大数据量不卡顿

### AI 查询
1. 在 AI 查询面板输入自然语言查询
2. 例如："帮我找一下 RT 地址 5 的所有数据"
3. 系统自动解析并筛选数据
4. 周期间隔分析：输入"显示终端地址26且子地址3的周期间隔"

### 甘特图/统计图
1. 在数据范围下拉框选择数据范围
2. 支持：文件所有数据/当前页数据/当前筛选数据
3. 切换后自动更新图表

### 周期间隔折线图
1. 通过 AI 查询输入"显示终端地址X且子地址Y的周期间隔"
2. 自动切换到图表选项卡，显示周期间隔折线图
3. 数据表格自动筛选为相同 RT 和子地址的数据
4. 折线图每个数据点标注数值
5. 点击"下载CSV"按钮导出详细数据（含序号、RT、子地址、数据包时间、间隔时间）
